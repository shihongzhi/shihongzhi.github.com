    Transition

  • 一个过程
  • 两个状态
  • 平滑过渡的过程控制

    • Transition

    
                        #box {
                            transition-property: width, height;
                            transition-duration: 2s;
                            height: 150px;
                            width: 150px;
                        }
                        #box:hover {
                            height: 500px;
                            width: 500px;
                        }

    Transition

    Transition

    属性

    
                        transition-property
                        transition-duration
                        transition-timing-function   //如何计算中间插值
                        transition-delay
                        transtion

    Animation

  • 一个过程
  • 两个状态
  • 动画阶段关键帧的明细控制---keyframes

    • Animation

    Animation

    
                        .box_animation:hover {
                          -webkit-animation: myanim 5s infinite;
                        }

                        @-webkit-keyframes myanim {
                          0%   { opacity: 0.0; }
                          50%  { opacity: 0.5; }
                          100% { opacity: 1.0; }
                        }
